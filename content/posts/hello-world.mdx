# 欢迎来到我的技术博客

这是我使用 Next.js 和 MDX 搭建的个人技术博客。在这里，我将分享：

- 编程心得
- 技术文章
- 代码示例
- 项目经验

## 博客特性

这个博客支持以下特性：

1. **Markdown 渲染** - 使用 MDX 支持 Markdown 和 JSX 混写
2. **代码高亮** - 支持多种编程语言的语法高亮
3. **暗黑模式** - 自动适配系统主题，支持手动切换
4. **标签分类** - 文章按标签分类，方便查找
5. **阅读进度** - 顶部显示阅读进度条
6. **目录导航** - 自动生成文章目录
7. **响应式设计** - 适配各种屏幕尺寸
